Flashcards

JavaScript

The most widely used scripting langauge on the web, the only language supporte by all browsers.

jQuery

The most popular JavaScript libray, useful for speeding up development time on web applications.

HTML

"Hypertext Markup Language" used for semantic markup, to describe content to a browser.

CSS

"Cascading Style Sheets" used to style the look and feel of a website.

Bootstrap

Developed by Twitter, this is the most popular CSS library today.

Git

A popular version control system.

GitHub

An online community for developers who use git to store and share their repositories.

Markdown

A markup language used for README files (on GitHub) and also popular for writing wikis.

Epicodus

An awesome web development bootcamp in Portlan, Oregon (and Seattle, WA - with more locations soon)

Codepen.io

A website developed to allow coders to practice building and publish websites.

PHP

A server-side scripting language commonly used to build websites and popular CMS apps like WordPress.

mySQL

An open source database popular for building websites.

C#

A Microsoft created programming language similar to Java.

Ruby

A programming language used for building web appliations.

MongoDB

Another open-source database commonly used with JavaScript for building websites.

Angular.js

A JavaScript library created to speed up development based on MVC.

MVC

"Model-View-Controller" MVC applications separate presentation, data and business rule layers.

SASS

A CSS preprocessor.

Drupal

A popular CMS (Content Managment System) used to build contnet-rich websites.

.NET

A framework developed by Microsoft to greatly speed up web development.
